# Automating Stuffs

Note that this will create a massive traffic in the server end and cause a smelly logs in the sql logging database. Hence this is quite a bad OPSEC to proceed with in the first page Above steps could be automated with sqlmap tool. Lets throw the request to sqlmap and see available databases. Unfortunately, we do not have any data in rabbit_spice database and we also doesn’t have a high privilege used to write or read file onto the server. This could be checked with user() built-in sql function. Enumerating privileges with --privilege flag with sqlmap also shows that we only have a USAGE privilege. So, here we go, another day another rabbit hole. Lets move on onto the next one.
